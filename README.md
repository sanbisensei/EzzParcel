# 📦 EzzParcel — Parcel Delivery Management System

EzzParcel is a web-based parcel delivery management system designed to streamline parcel sending, tracking, rider assignment, and payments — all within a single platform. The goal was not just to build an app, but to think like system designers and translate real-world delivery workflows into a structured information system.

---

## 🔹 Key Features

- 📦 Parcel creation & real-time tracking across Bangladesh
- 💳 Secure online payments using Stripe
- 👥 Role-based dashboards for **Customer**, **Rider**, and **Admin**
- 🧑‍💼 Admin rider approval & parcel assignment by district
- 💬 Feedback & chat system for better communication
- 🔐 Secure authentication with Firebase

---

## 🔹 Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React + Vite |
| Authentication | Firebase Authentication |
| Data Fetching | Axios & React Query |
| Payments | Stripe Payment Gateway |
| Animations | GSAP (basic) |
| UI Design | Neo-brutalist |
